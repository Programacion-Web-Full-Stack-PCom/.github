# Curso de Programación Web Full Stack Instituto PuntoCom

Este curso está enfocado en la formación de programadores full stack desde 0. Abarcando desde creación de páginas web (frontend), diseño de base de datos, hasta programación de servidores backdend.

## Herramientas

- VsCode: Editor de código integrado
- Git: Herramienta de control de versiones
- NodeJs: Framework de codigo javascript
- Reactjs: Framework de frontend
- PNPM: Gestor de paquetes
